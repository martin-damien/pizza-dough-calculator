# Pizza Dough Calculator

A simple desktop application to calculate pizza dough ingredients, built with Lazarus (FreePascal).

This tool helps you prepare your dough easily by calculating the right quantities of:
- Flour
- Water
- Salt
- Yeast

based on your recipe parameters.

---

## ✨ Features

- Calculate dough for any number of pizzas
- Adjustable dough weight per pizza
- Hydration control (% of water)
- Salt percentage
- Fermentation time and temperature
- Automatic yeast calculation
- Simple and intuitive interface

---

## 🧮 Yeast Calculation

The yeast amount is automatically calculated based on:
- fermentation duration
- temperature
- hydration
- salt

The goal is to provide **realistic and practical values** for home pizza making.

The result is given in **fresh yeast (grams)**.

## 🚀 How to use

1. Enter:
   - number of pizzas
   - dough weight
   - hydration (%)
   - salt (%)
   - fermentation time
   - temperature

2. Click calculate

3. The application will display all ingredient quantities
